---
title: Native X Articles
description: Draft in Quill, review the real private X Article, then schedule it.
---

Quill supports native X Articles with rich DraftJS content state, a cover asset, and referenced Quill media assets. An Article is not an ordinary post with a long caption.

## Review flow

1. Create the Article record with `create_article_draft`.
2. Call `create_article_review`. Quill uploads the referenced owned media and creates a private native X Article draft.
3. Open the returned `reviewUrl` on X and check the actual Article—the title, body, embeds, cover, and media.
4. Once you explicitly approve it, use `schedule_article` with an exact UTC time and named timezone.

Quill schedules the reviewed X Article ID. Creating a fresh review after edits produces a fresh private X draft, so you never accidentally publish an earlier reviewed version.

## Article media

Upload the owned file with `upload_media_asset`. Pass the returned asset ID as `coverAssetId` for the cover, or include asset IDs in `contentState.entities[*].value.data.asset_ids` for body media. Quill replaces those private asset IDs with X media IDs at review time.

Every tool is scoped to the bearer key's Quill user and X API connection.

## `list_media_assets`

**Read tool.** List the authenticated X account's durable uploaded image and video assets, ready to attach to a draft or X Article.

_No input._
## `upload_media_asset`

**Write tool.** Store an owned JPEG, PNG, WebP, GIF, MP4, or MOV asset in Quill. Pass base64 bytes. Quill uploads it to X only when its human-approved draft/article is reviewed or published.

```json title="Input schema"
{
  "type": "object",
  "properties": {
    "filename": {
      "type": "string",
      "minLength": 1,
      "maxLength": 160
    },
    "contentType": {
      "type": "string",
      "enum": [
        "image/jpeg",
        "image/png",
        "image/webp",
        "image/gif",
        "video/mp4",
        "video/quicktime"
      ]
    },
    "dataBase64": {
      "type": "string",
      "minLength": 4,
      "maxLength": 720000000,
      "description": "Raw file bytes base64 encoded; do not include a data URL prefix."
    }
  },
  "required": [
    "filename",
    "contentType",
    "dataBase64"
  ],
  "additionalProperties": false,
  "$schema": "http://json-schema.org/draft-07/schema#"
}
```
## `delete_media_asset`

**Destructive tool.** Delete an unreferenced media asset. Quill refuses deletion when an asset is attached to a draft, schedule, or Article.

```json title="Input schema"
{
  "type": "object",
  "properties": {
    "assetId": {
      "type": "string",
      "format": "uuid"
    }
  },
  "required": [
    "assetId"
  ],
  "additionalProperties": false,
  "$schema": "http://json-schema.org/draft-07/schema#"
}
```

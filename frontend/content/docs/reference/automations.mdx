**Write tool.** Create a CTA automation that posts the configured CTA reply after an owned X post reaches the chosen like threshold. Only use with the human's explicit approval.

```json title="Input schema"
{
  "type": "object",
  "properties": {
    "sourceXPostId": {
      "type": "string",
      "minLength": 1
    },
    "ctaText": {
      "type": "string",
      "minLength": 1,
      "maxLength": 25000
    },
    "likeThreshold": {
      "type": "integer",
      "exclusiveMinimum": 0
    }
  },
  "required": [
    "sourceXPostId",
    "ctaText",
    "likeThreshold"
  ],
  "additionalProperties": false,
  "$schema": "http://json-schema.org/draft-07/schema#"
}
```
